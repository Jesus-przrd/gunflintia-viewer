# Gunflintia Specimen Viewer

Standalone WebGL viewer for 3D reconstructions of Gunflintia microfossils, built as a
shareable demo of the specimen models produced for the evostitch research viewer at
Blue Marble Space.

Built with [three.js](https://threejs.org/) — loads glTF models, no build step, runs
as a static page.

**[Live demo](https://jesus-przrd.github.io/gunflintia-viewer/)**

## Features
- Drag to rotate, two-finger/right-click drag to pan, scroll to zoom
- Model selector to switch between specimen reconstructions (living and fossilized Gunflintia)
- Physically-based rendering with a neutral room environment for consistent lighting

## Tech
Three.js · GLTFLoader · OrbitControls
